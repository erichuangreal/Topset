# Mood avatar images (gym cat)

The Home page shows a cartoony gym cat avatar that changes with the mood indicator.

Required files in this folder (same directory as this file):

- `avatar-locked-in.png` – energetic, pumped gym cat
- `avatar-steady.png` – calm, focused gym cat
- `avatar-sore.png` – tired but satisfied gym cat
- `avatar-drained.png` – sleepy, low-energy gym cat

If these files are missing, the app falls back to the default user icon in the avatar card.

Generated images were saved under the Cursor project assets path; copy them here if needed.
